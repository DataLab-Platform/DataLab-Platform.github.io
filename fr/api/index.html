API#

L’interface de programmation d’application (API) publique de DataLab offre un ensemble de fonctions pour accéder aux fonctionnalités de DataLab. Ces fonctions sont disponibles dans divers sous-modules du package cdl. Le tableau suivant répertorie les sous-modules disponibles et leurs contenus respectifs :

Sous-module

Contenu

cdl.algorithms

Algorithmes pour l’analyse de données, qui opèrent sur des tableaux NumPy

cdl.param

Module de commodité pour accéder aux ensembles de paramètres DataLab (instances d’objets guidata.dataset.DataSet)

cdl.obj

Module de commodité pour accéder aux objets DataLab (cdl.obj.SignalObj ou cdl.obj.ImageObj) et fonctions associées

cdl.core.computation

Fonctions de calcul, qui opèrent sur les objets DataLab (cdl.obj.SignalObj ou cdl.obj.ImageObj)

cdl.proxy

Objets proxy pour accéder à l’interface DataLab à partir d’un script Python ou d’une application distante
